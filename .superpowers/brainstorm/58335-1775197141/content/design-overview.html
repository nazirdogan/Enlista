4. Architecture & Data Flow

New Route: /contact-sales → app/(public)/contact-sales/page.tsx
New API: POST /api/contact-sales
  • Validates form fields (server-side)
  • Inserts into contact_leads table in Supabase
  • Sends branded HTML email to nazir@enlista.io via Resend
  • Returns success/error response
New DB Table: contact_leads
  • id, first_name, last_name, email, phone
  • agency_name, employee_count, location
  • focus_area (text[]), message
  • created_at, status (new/contacted/converted)
